Protected Endpoints

MethodPathRequired Authority
GET/api/v1/secure/meAuthenticated token
GET/api/v1/secure/employeeROLE_EMPLOYEE or above
GET/api/v1/secure/managerROLE_MANAGER or ROLE_ADMIN
GET/api/v1/secure/adminROLE_ADMIN

Token Rules

  • Security filter accepts only Authorization: Bearer <access-token> for endpoint authorization.
  • Refresh token is accepted only by refresh/logout APIs and must carry claim type=refresh.
