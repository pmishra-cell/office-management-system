Track usage of methods, variables, or log lines across the source code. Results show file path, Java package, exact line number, and line content.

How to run this page:
Start server from project root with python3 -m http.server --cgi 8000
Then open http://127.0.0.1:8000/docs/code-search.html
If you see Python source at /cgi-bin/code_search.py, the server is running without CGI. Stop it and restart with --cgi.
