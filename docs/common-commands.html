Code Search Documentation

Serve Docs in Browser

python3 -m http.server 8000

Serve Docs With Code Search (CGI)

python3 -m http.server --cgi 8000
http://127.0.0.1:8000/docs/code-search.html

Build and Test

mvn clean package
mvn test

Run Application

mvn spring-boot:run

Run Application With Redis Security Mode (One Command)

./docs/run-local-redis.sh

This starts Redis using Docker Compose and then runs Spring Boot with the local-redis profile.

Stop Local Redis Service

./docs/stop-local-redis.sh

Manual API Calls

curl -i -X POST http://127.0.0.1:8080/api/v1/auth/login \\
  -H "Content-Type: application/json" \\
