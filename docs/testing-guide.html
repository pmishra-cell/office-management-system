What Is Covered

  • Register/login/refresh/logout API flows
  • Refresh token rotation and revocation
  • Lockout policy and 429 behavior
  • Role-based endpoint authorization
  • JWT token claim/type behavior
  • In-memory store expiration and CRUD semantics

Run Tests

mvn test

Run Coverage (JaCoCo)

mvn clean verify

Coverage report output:

  • target/site/jacoco/index.html

Testing Notes

  • Integration tests use Spring Boot test context with in-memory DB defaults.
  • Redis-specific behavior is implemented through conditional stores and can be manually verified using local Redis profile scripts.
